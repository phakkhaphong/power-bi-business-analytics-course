## [1.0.0] - 2024-10-13

### Added
- Initial release of Power BI Business Analytics Course
- Complete course structure with 15 lessons
- README.md with comprehensive course overview
- CONTRIBUTING.md for instructors
- requirements.md with system requirements
- .gitignore for Power BI projects
- .gitattributes for proper file handling
- LICENSE file (MIT License)

### Course Structure
- **01-Introduction**: Introduction to Power BI
- **02-Desktop-Overview**: Power BI Desktop Overview
- **03-Data-Sources**: Data Sources and Connections
- **04-Power-Query**: Power Query Editor
- **05-Data-Transformation**: Data Transformation
- **06-Data-Modeling**: Data Modeling
- **07-Visualizations**: Visualizations
- **08-Advanced-Visuals**: Advanced Visualizations
- **09-Report-Design**: Report Design
- **10-DAX-Fundamentals**: DAX Fundamentals
- **11-DAX-Functions**: DAX Functions
- **12-Advanced-DAX**: Advanced DAX
- **13-Power-BI-Service**: Power BI Service
- **14-Sharing**: Sharing and Collaboration
- **15-Best-Practices**: Best Practices

### Documentation

### Resources
